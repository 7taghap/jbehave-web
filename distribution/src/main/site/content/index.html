What is JBehave Web?

JBehave Web is an extension of JBehave framework providing a web integration layer for Behaviour-Driven Development (BDD).

Features of JBehave Web include:

  • A Web Queue to allow generic stories to be run asynchronously via a simple web interface.
  • A Web Runner to allow generic stories to be run synchronously via a simple web interface.
  • A Selenium integration module to allow automation of stories for web applications using Selenium.
